## 🎯 Objetivo do Software

O sistema **CENAFLIX** tem como objetivo permitir o gerenciamento de conteúdos de uma plataforma fictícia de streaming, possibilitando o cadastro, consulta e controle de informações relacionadas aos conteúdos disponíveis.

O software foi desenvolvido para fins acadêmicos, aplicando conceitos de **programação em Java**, **integração com banco de dados MySQL** e **controle de versão com GitHub**.
